１、一度に複数の範囲のセルを選択する方法

　基本的には、"shift"キー、"ctrl"キーを使ってセルを選択する方法を知っていれば大抵のことはできますが、[編集]メニューの[ジャンプ...]を使うとさらに細かいことができます。[ジャンプ...]については、Excelのヘルプの
『空白セル、または数式、定数、コメントを含むセルだけ選択する』、
と
『[選択オプション] ダイアログ ボックスのオプション』
を参照してください。

２、（アドインの実行後、）可視セルのみを選択する方法

　[編集]メニューの[ジャンプ...]を使うことにより、実行することができます。詳しくは、Excelのヘルプの
『可視セルのみを選択する』
を参照してください。
